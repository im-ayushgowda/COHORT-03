# Sync Landing Page

A modern and responsive landing page built using **HTML5** and **CSS3**. The project recreates a clean UI inspired by a design template using CSS Grid, Flexbox, and responsive scaling techniques.

## 🚀 Live Demo

Add your GitHub Pages link here:

https://im-ayushgowda.github.io/COHORT-03/ASSIGNMENT%20-%2003/

---

## 📌 Features

- Responsive layout using CSS Grid
- Modern UI design
- Flexbox for content alignment
- Responsive typography using `clamp()`
- CSS Container Queries (`cqw`) for scalable layouts
- Smooth scaling across different screen sizes
- Remix Icons integration
- Clean and well-structured code
